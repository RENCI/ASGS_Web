About the ASGS Operational Awareness Dashboard.

The ASGS Operational Awareness Dashboard is a web site that works in tandem with the Advanced Surge Guidance System (ASGS) to monitor compute resources analyzing storm data.


Shown below is a single ASGS instance panel.

Where:

  • A - The ASGS Site and run name.
  • B - The current state of the ASGS process instance.
  • C - The ASGS process instance parameters.
  • D - The Advisory progress area containing progress information for an advisory.
  • E - The progress bar displays information about the main ASGS process, sub-processes and textual information.
  • E - The message area displays event messages embedded within the ASGS code. Click to display the last 7 messages.
